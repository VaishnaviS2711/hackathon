# Hackathon

**problem**

Traditional studying from notes can become repetitive and less engaging for students. Many students struggle to stay focused and retain information while preparing for exams.

**how to run**

*Step 1 - The user enters:*

Study notes

Optional word limit

Important topics/questions (optional)

*Step 2 - AI Processing*

Reads the provided notes

Identifies important concepts and keywords

Generates MCQ questions

Creates four answer options

Determines the correct answer

*Step 3 - Game Mechanism*

A character runs on a path

Four gates appear with answer options

The player selects the gate with the correct answer

Correct answer → score increases and game continues

Wrong answer → game ends

*Step 4: Result Generation*

Final score

Number of correct answers


**What AI does**
It generates questions based on the notes provided and adds point based on which questions we get right.
